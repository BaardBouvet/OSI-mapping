# Nested Arrays - Multiple Branches

Demonstrates complex nested structure with multiple independent branches from a single root.

## How it works

1. Organization has two independent nested arrays: `departments` and `projects`
2. Departments contain nested `employees`; projects contain nested `tasks`
3. Creates 5 targets: `organization`, `department`, `employee`, `project`, `task`
4. Both branches go two levels deep with independent reference chains
